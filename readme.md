## scoop-bucket

[Scoop](https://scoop.sh/) bucket for:

- [PRM](https://github.com/ldez/prm) binaries.

```bash
scoop bucket add prm https://github.com/ldez/scoop-bucket.git
scoop install prm
```

- [motoko](https://github.com/ldez/motoko) binaries.

```bash
scoop bucket add prm https://github.com/ldez/scoop-bucket.git
scoop install motoko
```
